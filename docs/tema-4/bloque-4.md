# Bloque 4. Análisis de datos cualitativos

...

## Ideas clave del bloque

- El análisis cualitativo es un proceso interpretativo y sistemático.
- Se desarrolla de forma continua y cíclica.
- La reducción, organización e interpretación de datos son fases clave.
- Las categorías estructuran el significado de la información.
- Los programas informáticos apoyan, pero no sustituyen, al investigador.
- El rigor se basa en la coherencia y la transparencia del proceso.
