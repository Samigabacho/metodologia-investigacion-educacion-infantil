# Generador de datos

A través de esta herramienta podéis generar matrices de datos ajustadas a vuestro diseño de investigación.

---

## Uso de la plantilla

1. Accede a la hoja.
2. Haz una copia en tu Drive.
3. Modifica los parámetros según tu diseño.
4. Descarga los datos o úsalos para el análisis.
